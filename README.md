# Dashboard de Análise de Churn

Análise de cancelamento de clientes usando Python

## Funcionalidades

- KPIs principais (total de clientes, taxa de churn, receita perdida)
- Análise de atraso de pagamento
- Análise de contatos com call center
- Insights automáticos

## Tecnologias e libs

- Python 3.9+
- Streamlit
- Pandas
- Plotly Express

## Como rodar

```bash
pip install -r requirements.txt
streamlit run app.py
```
